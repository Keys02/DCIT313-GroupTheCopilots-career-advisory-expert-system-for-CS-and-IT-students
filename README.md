# DCIT313-GroupTheCopilots-career-advisory-expert-system-for-CS-and-IT
A Career Advisory Expert System is an intelligent computer system designed to help students choose suitable career paths based on their skills, interests, and academic performance. It works by using expert knowledge about different careers and representing that knowledge in the form of rules.

The system asks students questions about their abilities, preferences, and strengths in areas such as programming, data analysis, networking, or problem-solving. Using these responses, the system applies a set of IF–THEN rules stored in its knowledge base to analyze the information and determine which careers best match the student’s profile.

The expert system typically includes a user interface for interaction, a knowledge base containing career-related rules, an inference engine that processes the rules, and an explanation module that explains why a particular career was recommended. The goal of the system is to provide reliable and structured career guidance to students and assist them in making informed decisions about their future professions.
